This repository contains the hands-on exercises, configurations, scripts, documentation, and projects completed during the JK Tech Multi-Cloud DevOps Training.

The repository is organized into separate folders based on different DevOps technologies and training activities. Each folder will be updated as new exercises and practical tasks are completed.

Repository Structure devops-training-alan/ │ ├── docs/ ├── scripts/ ├── .github/ │ └── workflows/ ├── docker/ ├── terraform/ ├── kubernetes/ ├── aws/ ├── azure/ ├── gcp/ ├── runbooks/ ├── capstone/ └── README.md Folder Description docs/ Contains training notes, reference documents, diagrams, and supporting documentation.

scripts/ Contains Bash, PowerShell, and automation scripts developed during the training.

.github/workflows/ Contains GitHub Actions workflow files for Continuous Integration and Continuous Deployment (CI/CD).

docker/ Contains Dockerfiles, Docker Compose files, and container-related exercises.

terraform/ Contains Infrastructure as Code (IaC) configurations for provisioning cloud resources.

kubernetes/ Contains Kubernetes manifests, deployments, services, ConfigMaps, Secrets, and other cluster resources.

aws/ Contains hands-on exercises, configuration files, and examples related to Amazon Web Services.

azure/ Contains Azure resource configurations, scripts, and practical exercises.

gcp/ Contains Google Cloud Platform configurations, scripts, and hands-on exercises.

runbooks/ Contains operational procedures, troubleshooting guides, and standard operating procedures (SOPs).

capstone/ Contains the final project developed by integrating the technologies covered during the training.

Hands-on Activities This repository will be used to maintain:

Git and GitHub practice Feature branch development Pull Requests and code review updates Repository documentation Automation scripts Infrastructure as Code Containerization Kubernetes deployments Cloud platform exercises CI/CD workflow implementations Capstone project Current Status Status: Training in Progress

The repository will be updated continuously with practical exercises, scripts, workflow files, documentation, and project deliverables as the training progresses.

Prerequisites Before starting, install: - Git - VS Code - WSL (Windows users) - GitHub account
